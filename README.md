# Website Project

## Overview
This project demonstrates the use of Git branching, merging, and conflict resolution in a collaborative development workflow.

The project includes:
- Navigation Bar
- Hero Section
- Footer
- Contact Form
- Testimonials Section

## Technologies Used
- HTML
- Git
- GitHub
- VS Code

## Features Implemented

### Navigation Bar
Added links for:
- Home
- About
- Contact

### Hero Section
Added a welcome section with a heading and short description.

### Contact Form
Created a form with:
- Name field
- Email field
- Message field

### Testimonials Section
Added customer testimonial placeholders.

### Footer
Added a footer section with copyright information.

## Git Workflow Practiced
- Creating branches
- Switching branches
- Merging branches
- Resolving merge conflicts
- Pushing code to GitHub
